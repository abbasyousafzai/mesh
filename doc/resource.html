class mesh.resource.Configuration(standard_requests=None, validated_requests=None, id_field=None, default_controller=None)[source]¶

A resource configuration scheme.

Parameters:
  • standard_requests (dict) – Optional, default is None; a dict mapping request names to callables which will generate a standard request for a given resource. The callables should take a single Resource argument.
  • validated_requests (list) – Optional, default is None; a list indicating which standard requests provided by this configuration should be validated by validators which don’t explicitly specify requests.
  • id_field (Field) – Optional, default is None; a mesh.schema.Field which will serve as the unique identifier field for resources associated with this configuration which do not themselves declare an identifier field. If not specified, a mesh.schema.Integer field with a name of id will be constructed for this configuration.
  • default_controller (class) – Optional, default is None; a subclass of Controller which will be used as the base class for automatically generated mock controllers under this configuration.

class mesh.resource.Resource[source]¶

A resource definition.

name¶

The name of this resource; should either be None for abstract resources or a string of lowercase letters for concrete resources.

version¶

The version of this resource definition; should either be None for abstract resources or an integer >= 1 for concrete resources.

schema¶

A dict mapping field names to mesh.schema.Field instances which together comprise the schema of this resource.

requests¶

A dict mapping request names to mesh.request.Request instances declaring the possible requests for this resource.

validators¶

A dict mapping validator names to the custom validators declared for this resource. Not intended for public use.

description¶

An optional description of this resource, pulled from the docstring of the class if present. Used primarily for documentation.

maximum_version¶

A class property which indicates the version number of the newest definition of this resource.

minimum_version¶

A class property which indicates the version number of the oldest definition of this resource.

class mesh.resource.Controller[source]¶

A resource controller.

resource¶

The resource this controller implements; should either be None for abstract controllers or a Resource definition for concrete controllers.

version¶

The version of this controller; should either be None for abstract controllers or a tuple containing two integers for concrete controllers: the version of the resource being implemented followed by the minor version within that version for this controller.

maximum_version¶

A class property which indicates the version tuple of the newest definition of this controller.

minimum_version¶

A class property which indicates the version tuple of the oldest definition of this controller.
