mesh by storediq

a declarative RESTful API framework

This is Mesh, a RESTful API framework for Python and Javascript.

Dependencies

Mesh builds on Scheme -- our schema framework.

Install

Mesh and Scheme are up on PyPI and should work fine with easy_install: 

$ easy_install scheme
$ easy_install mesh

  • Mesh on PyPI
  • Scheme on PyPI
